# Pochacco Birthday Surprise - 结婚一周年快乐

这是一个基于 Three.js 和 MediaPipe 的互动网页，用于庆祝结婚一周年。

## 主要功能

*   **3D 粒子系统**：由粒子组成的帕恰狗（Pochacco）形象。
*   **手势识别互动**：使用摄像头识别手势进行互动。
    *   **✊ 握拳 (FIST)**：召唤/重组帕恰狗。
    *   **🖐 张手 (PALM)**：粒子爆散效果。
    *   **🤏 捏合 (PINCH)**：粒子变形为预设的合照（按顺序切换）。
    *   **👋 挥手 (MOVE)**：旋转视角。
*   **预设合照**：系统会自动加载 `images/photos/` 目录下的照片（1.jpg, 2.jpg）。

## 🖼️ 如何添加更多合照 (非常重要)

这个网页使用了**自动序号嗅探**技术，你不需要修改任何代码即可添加无限多的照片。

1.  找到项目文件夹下的 `images/photos/` 目录。
2.  将你的合照放入该文件夹。
3.  **必须**按照数字顺序重命名文件，格式必须是 `.jpg`：
    *   `1.jpg`
    *   `2.jpg`
    *   `3.jpg`
    *   `4.jpg`
    *   ...以此类推
4.  **注意**：序号必须连续，中间不能断开。如果只有 `1.jpg` 和 `3.jpg`，系统读到 `2.jpg` 失败后就会停止，`3.jpg` 将不会被显示。

## 文件结构

*   `index.html`: 入口文件。
*   `js/`: 
    *   `main.js`: 主逻辑，负责初始化和资源加载。
    *   `particle-system.js`: 粒子系统核心，包含变形和物理运动逻辑。
    *   `gesture-control.js`: 手势识别处理。
*   `images/photos/`: 存放合照图片 (1.jpg, 2.jpg)。

## 注意事项

*   请确保浏览器允许使用摄像头权限。
*   本地运行时，建议使用简单的 HTTP 服务器（如 VS Code 的 Live Server）以避免跨域问题。
